Come si gioca

Un draft alla cieca, match dopo match

Decempionz è un gioco di blind draft: scegli una formazione, poi ti vengono proposti tre giocatori alla volta — uno va nel tuo XI, gli altri spariscono. Niente certezze, solo intuito e strategia.

1

Scegli il torneo e il formato

UCL, Copa o World Cup. Classic (girone + playoff), Old Cup (solo knockout) o New Wave (fase a lega UCL). Difficoltà facile, normale o difficile.

2

Scegli un'era storica

Anni '60, '70, '80, '90, 2000s, 2010s oppure All Time per pescare da ogni epoca. Ogni era ha il suo pool di squadre e giocatori.

3

Fai il draft del tuo XI

Formazione a scelta (4-3-3, 4-4-2, 3-5-2 e altre), poi un draft a tier: le 3 carte mostrate appartengono sempre allo stesso livello di rating (★★★★ Elite, ★★★ Alto, ★★ Medio, ★ Base). Usa il reroll per cambiare tier — ma in modalità Normale e Difficile è bloccato negli ultimi 3 slot. I giocatori ★ (9.5+) danno un bonus xG.

4

Gioca il torneo

Ogni partita è simulata con meccaniche di momentum, tattica e scout report avversario. Pareggi ai 90? Si va ai rigori.

Aspetti tecnici

Un gioco in un singolo file HTML

Decempionz è costruito in JavaScript vanilla, senza framework, senza dipendenze esterne, senza database. Un file HTML di circa 6.000 righe che contiene tutto: logica, dati storici, grafica e animazioni.

Funziona su qualsiasi browser moderno, si carica in meno di un secondo e può essere salvato e giocato anche offline come PWA. È un esperimento di game design minimalista: quanta profondità si può mettere in un singolo file?

Il codice è open source su GitHub per chi vuole esplorarlo, contribuire o fare un fork per il proprio torneo storico preferito.

How to play

A blind draft, match by match

Decempionz is a blind draft game: pick a formation, then three player cards appear at once — choose one for your XI, the others disappear. No guarantees, only instinct and strategy.

1

Choose tournament and format

UCL, Copa or World Cup. Classic (group stage + playoffs), Old Cup (pure knockout) or New Wave (league phase, UCL only). Easy, Normal or Hard difficulty.

2

Pick a historical era

1960s, 1970s, 1980s, 1990s, 2000s, 2010s — or All Time to draw from every era.

3

Draft your XI

Choose a formation (4-3-3, 4-4-2, 3-5-2 and more), then fill every slot through a tiered draft: all 3 cards always share the same rating level (★★★★ Elite, ★★★ High, ★★ Mid, ★ Base). Reroll to change tier — but on Normal and Hard it's locked for the last 3 slots. Players rated ★ (9.5+) give an xG shooting bonus.

4

Play the tournament

Each match is simulated with momentum mechanics, tactics and an opponent scout report. Draw after 90 minutes? Penalty shootout.

Technical

An entire game in one HTML file

Decempionz is built in vanilla JavaScript — no frameworks, no external dependencies, no database. A single ~6,000-line HTML file containing everything: game logic, historical data, graphics and animations.

It runs in any modern browser, loads in under a second, and can be installed and played offline as a PWA. It's an exercise in minimalist game design: how much depth can you pack into a single file?

The source code is open source on GitHub for anyone who wants to explore it, contribute, or fork it for their own favourite historical tournament.
